# react_ts_todolist
咖玛todolist需求


- 显示任务列表
- 添加新任务（含校验）
- 标记任务为完成 / 切换
- 删除任务（含确认）
- 按状态过滤（全部 / 待办 / 已完成）
- 使用 `localStorage` 持久化
- 响应式布局（窄屏自适应）

## 环境要求
- Node.js 16+ 或更高
- npm 或 yarn

## 安装与运行
```bash
# 克隆仓库
git clone [your-repo] task-manager
cd react_ts_todolist

# 安装依赖
npm install

# 本地开发(vite)
npm start
# 打开 http://localhost:80
